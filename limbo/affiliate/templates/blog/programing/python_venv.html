How to create python virtual Environment

By Rohit Raj- Updated on 23 July, 2020

Now a day python become most popular high level language. Python was created by Guido Van Rossum and released in 1991. In this entire article i will teach you how to create python virtual Environment ?. If you not install python IDE in your desktop yet, then installed pyhton from here and choose python step up according to your OS.

What is virtualenv python ?

A virtual environment is a tool that can create a separate environment for each different project, which has no connection to the remote project. It is one of the most important tools that most Python developers use.

How does a virtual environment work?

We use a module named Virtualenv to create a separate virtual environment. After that we install the python package according to our project in that virtual environment.

Step #1. Installing virtualenv in window.

To install virtualenv in your system go to system terminal then type a command which is given below:

C:\Users\ASUS>pip install virtualenv


Step #2. Create a new folder.

After Installing virtualenv successfully, then create a new folder where to create virtual environment for your project. In my case will create a new folder in desktop and folder name is python venv. to setup virtual environment type a command which is given below:

C:\Users\ASUS>desktop>python venv>virtualenv myenv

Note : In my case, I gave the name 'myenv' and you can give some other names as well.


step #3: activate virtualenv in window.

After creating successfully virtual environment, you need to activate virtual environment. This can be done using the following command which is given below:

C:\Users\ASUS>desktop>python venv>myenv\Scripts\activate

After activating the virtual environment, your terminal should look something like this.

(myenv) C:\Users\ASUS>desktop>python venv>

You also need to know how to deactivate virtual environment by typing simple deactivate command.

(myenv) C:\Users\ASUS>desktop>python venv>deactivate


step #4: Installing Python package in virtual environment.

(myenv) C:\Users\ASUS>desktop>python venv>pip install Django==3.0.8

The Django 3.0.8 package will be placed in myenv folder and will be isolated from the complete system. In this way you can install python packages which is need for your project. If don't know about django framework Click here to read top 10 backend frameworks.

Note : You can also check install python package in your virtual environment by using freeze command.

(myenv) C:\Users\ASUS>desktop>python venv>pip freeze
